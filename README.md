# FaeresDevWebsite

## Install 

```
npm install react react-dom
npm create vite@latest faeresdev-site -- --template react
cd faeresdev-site
npm install
npm install antd
npm install react-router-dom
npm install tailwindcss @tailwindcss/vite
npm install @reduxjs/toolkit
npm install react-redux
npm install react-bootstrap
```

do not use beta, use npm and start now 

## to run test env

```
cd faeresdev-site
npm run dev
```


Documentations to help dev : 

https://ant.design/ (for ui components)
